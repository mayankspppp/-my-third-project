# 📊 Advanced Business Analytics Dashboard

This project is a real-world inspired Data Analytics Dashboard built using HTML, CSS, JavaScript, and Chart.js.

## 🚀 Features
- Real-time KPI tracking (Sales, Profit, Customers)
- Interactive charts using Chart.js
- Business insights visualization
- Clean Power BI-style UI
- Ready for GitHub Pages deployment

## 📈 Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- CSV data structure

## 🎯 Business Value
This dashboard simulates real-world business intelligence systems used in companies to analyze performance and make data-driven decisions.
